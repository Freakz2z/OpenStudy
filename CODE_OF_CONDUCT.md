# Code of Conduct

## Our Standard

OpenStudy is an open-source learning product, and the collaboration environment should feel constructive, respectful, and safe.

Examples of behavior that help:

- Giving clear, good-faith technical feedback
- Asking questions with curiosity instead of hostility
- Respecting different experience levels and communication styles
- Keeping disagreements focused on product or code decisions
- Sharing context when something is risky, broken, or uncertain

Examples of unacceptable behavior:

- Personal attacks, insults, or harassment
- Dismissive or demeaning language
- Repeated bad-faith arguments or intentional disruption
- Publishing private information without permission
- Sexualized language or unwanted advances

## Scope

This applies in issues, pull requests, discussions, and any other public collaboration spaces for this repository.

## Enforcement

Repository maintainers may edit, remove, or reject contributions and discussions that violate this code of conduct.

If you experience or witness harmful behavior, please open a private report through the repository owner if possible.

## Interpretation

When in doubt, optimize for kindness, clarity, and collaboration.
